# Git artefacts 🗂️

This directory contains contributor rules for Git-managed artefacts and CI/CD pipeline conventions.
Its scope is how branches, commits, pull requests, and the CI/CD pipeline MUST be structured and used in this repository.

- [remotes.md](remotes.md): reproducible `origin`/`fork` setup for the maintainer's fork-based workflow.
- [branch.md](branch.md), [commit.md](commit.md), [pull-request.md](pull-request.md), [pipeline.md](pipeline.md): rules for each artefact type.
